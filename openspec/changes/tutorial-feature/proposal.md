## Why

ポーカーチップ計算機は ScrollableCounter のドラッグ操作やチップ編集ダイアログなど、初見では分かりにくいインタラクションが複数ある。初回アクセス時にチュートリアルを表示することで、ユーザーが主要機能を迷わず使い始められるようにする。

## What Changes

- 初回アクセス時にステップバイステップのチュートリアルオーバーレイを表示する
- 各ステップで対象要素をハイライトし、説明テキストをポップオーバーで表示する
- チュートリアルはスキップ可能（「スキップ」ボタン）
- ヘッダーにヘルプボタン（`?` アイコン）を追加し、いつでもチュートリアルを再表示可能にする
- チュートリアル完了バージョンを localStorage に保存し、バージョンが上がった場合は再度チュートリアルを表示する

## Capabilities

### New Capabilities
- `tutorial`: 初回チュートリアルオーバーレイの表示制御、ステップ管理、ハイライト、バージョン管理付き localStorage 永続化

### Modified Capabilities
- `ui`: ヘッダーにヘルプボタンを追加し、チュートリアル対象要素に `data-tutorial-step` 属性を付与する

## Impact

- `src/components/PokerChipCalculator.tsx`: ヘルプボタン追加、チュートリアル対象要素へのマーカー属性付与
- 新規コンポーネント: チュートリアルオーバーレイ、ステップポップオーバー
- localStorage: `tutorial-version` キーの追加（完了したチュートリアルのバージョン番号を保存）
- 新規依存: なし（既存の shadcn/ui Dialog + CSS で実装可能）
