## Why

新しいハンドやセッション開始時に、各チップを個別に0に戻す操作は手間がかかる。ワンタップで全チップの枚数を0にリセットできれば、チップ構成（額面・色）を維持したまま素早く新しいカウントを始められる。

## What Changes

- Chips セクションヘッダーに「Reset」ボタン（ghost variant）を追加する
- タップで全チップの `count` を 0 にリセットする（チップの額面・色・並び順は維持）
- 確認ダイアログは表示しない（枚数は即座に手動で戻せるため影響が軽い）

## Capabilities

### New Capabilities

なし（既存 UI の拡張のみ）

### Modified Capabilities

- `ui`: Chips セクションヘッダーにリセットボタンを追加し、全チップの枚数を一括で 0 にする操作を提供する

## Impact

- **変更ファイル**: `src/components/PokerChipCalculator.tsx`（リセットボタンの追加、リセットハンドラの実装）
- **依存関係**: 追加なし（既存の shadcn/ui Button で実装可能）

## Cross-Feature Dependencies

- **Presets ボタンとの配置**: Chips セクションヘッダーに Reset ボタンを追加すると、既存の Presets ボタンと並ぶ。モバイルでの配置バランスに注意。Reset はアイコンのみ（`RotateCcw`）にすることも検討
- **チュートリアル**: リセットボタン追加のためにチュートリアルステップを追加する必要はないが、チップカウンターのステップ説明に「リセットでゼロに戻せます」と軽く言及する選択肢あり
