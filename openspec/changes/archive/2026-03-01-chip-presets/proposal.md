## Why

現状、チップの初期値は `100 × 10枚` の固定構成であり、ゲーム開始時に毎回手動でチップを1行ずつ追加・設定する必要がある。キャッシュゲーム（1/2, 1/3, 2/5 等）やトーナメント（5K, 30K スタート等）では使用するチップ構成が大きく異なるため、プリセットから一括でチップ構成とブラインドを復元できると、セットアップの手間が大幅に削減される。

## What Changes

- **ビルトインプリセット**: ポーカーで一般的なキャッシュゲーム/トーナメントのチップ構成をアプリにバンドルして提供する（1/2 NL, 1/3 NL, 2/5 NL, 5/10 NL, Home Tournament 5K, Standard Tournament 30K 等）
- **ユーザーカスタムプリセット**: 現在のチップ構成+ブラインド設定をプリセットとして localStorage に保存し、後から呼び出せるようにする
- **プリセット読み込み**: プリセット選択時にチップリスト全体とブラインド値を一括で置き換える
- **プリセット管理**: ユーザー保存プリセットの名前変更・削除機能

## Capabilities

### New Capabilities
- `chip-presets`: プリセットの定義・保存・読み込み・管理に関するすべての機能を包括する。ビルトインプリセットのデータ定義、ユーザーカスタムプリセットの CRUD、プリセット選択 UI を含む。

### Modified Capabilities
- `ui`: Chips セクションにプリセット選択 UI を追加する。ヘッダー行にプリセットボタンを配置し、プリセット一覧ダイアログを提供する。

## Impact

- **新規ファイル**: `src/lib/presets.ts`（プリセットデータ定義・ロジック）、プリセット選択 UI コンポーネント
- **変更ファイル**: `PokerChipCalculator.tsx`（プリセット読み込み機能の統合、状態更新ロジック）
- **ストレージ**: localStorage に `chip-presets` キーでユーザープリセットを永続化（既存の `chips` sessionStorage / `stack-session` localStorage とは独立）
- **依存関係**: 追加なし（既存の shadcn/ui Dialog, Button で実装可能）
